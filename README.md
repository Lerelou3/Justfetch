# Justfetch

A lightweight system information fetcher written in pure Python, inspired by [pfetch](https://github.com/dylanaraps/pfetch), [fastfetch](https://github.com/fastfetch-cli/fastfetch), and [neofetch](https://github.com/dylanaraps/neofetch).

Justfetch is an amateur project created for fun that fetches and displays system information in a clean, minimal format. It's designed to be simple, fast, and dependency-free (apart from Python itself).
